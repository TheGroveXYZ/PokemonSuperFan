# PokemonSuperFan
My First Front-End Website

This project is my first full front-end website built with HTML, CSS, and vanilla JavaScript. It was created as part of my high-school web development course, where I focused on learning core JavaScript concepts, interactive UI design, and the basics of user-centered development.

Features

Fully responsive front-end

Interactive elements built with vanilla JS

Clean, minimal UI inspired by beginner UX principles

Full development process documented (PDF included)

UX testing and feedback completed with 5 users, with summarized findings (PDF included)
